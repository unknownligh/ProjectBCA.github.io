Library

About Library

The Library contains 32506 volumes of books. The Library subscribes to 50 National and International Journals and 250 e-journals, 17 Newspapers besides a holding of 350 bound volumes of Journals. Books are classified and arranged according to the Dewey decimal classification scheme (DDC). Books are catalogued according to the Anglo American Catalogue Rule – II (AACR –II). The Library also contains CD-ROMs. The Library is open on all working days from 8.30 a.m. to 5.00 p.m.

Automation

Our Library services are fully automated through using the standard Library Software Package called “Autolib”.The circulation system is fully computerized and all the documents are Bar-coded. The QS600 Scanner is used for charging and discharging the books.

Access to Computerized Library Catalogue (OPAC)

The Library has a computerized catalogue, which provides online access to the users. It is helps to the users to search the documents by Author, Title, Subject, Keyword, and Publishers etc. The mode of Catalogue is OPAC – On – Line Public Access Catalogue and we have at present provided twoNodes in the Reference apart from this user can access throughout the campus.

Open Access System

The Library has adopted the Open Access System the users can go to the stacks, browse and select the books and other sources freely. The reading materials are arranged as per the Dewey decimal classification. Shelf guides list are provided at the stack racks. The users can easily find the books as per their need.
